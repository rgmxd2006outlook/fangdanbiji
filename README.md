# 防弹笔记法网页系统（M1）

这是根据《防弹笔记法》需求实现的首版前端工程，覆盖 M1 核心能力：
- 灵感收集箱（录入、展示、删除、每日提醒、30天清理）
- 核心任务库（创建、编辑、行动管理、灵感转行动）
- 经验沉淀库（完成归档、经验展示）
- 全局基础搜索（多关键词空格分隔）
- LocalStorage 本地持久化

当前已补充 M2 能力：
- 暂存收集箱（期限、提醒、转化任务行动、关联任务、自动清理）
- 标签系统（预设标签 + 自定义标签，任务/行动最多 3 标签）
- 覆盘功能（任务执行 7 天提醒，覆盘记录保存）
- 首页简易趋势图（近 7 天灵感/任务/归档）

## 启动方式

```bash
npm install
npm run dev:api
npm run dev
```

访问：`http://localhost:5173`

说明：
- `npm run dev` 启动前端
- `npm run dev:api` 启动本地备份接口（`http://localhost:8787`）
